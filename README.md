# FastAPI — время сервера

Бэкенд возвращает текущее время сервера.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
uvicorn main:app --reload
```

Сервер: http://127.0.0.1:8001  
Документация: http://127.0.0.1:8001/docs

## Эндпоинты

| Метод | Путь   | Описание              |
|-------|--------|------------------------|
| GET   | `/`    | Приветствие            |
| GET   | `/time`| Текущее время сервера  |
